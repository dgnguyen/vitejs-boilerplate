# Vitejs + React + TypeScript + Rudux Thunk + Material UI Boilerplate

### Setup Project

```
npm i
```

### Run local server

```
npm start
```

### Build Application

```
npm run build
```

### ENV

Please include a .env in the root folder in below format

```
VITE_API_BASE_URL='http://localhost/api'
```
